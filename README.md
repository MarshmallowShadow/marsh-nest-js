# Marsh NestJS

Read NestJS basic setup README [here](./docs/README.md)


## Project Setup

``` bash
$ npm install
```

## Run Options

``` bash
$ npm run start

$ npm run start:dev

$ npm run start:prod
```

## Specs

- Runtime: Node v22.14.0
- Framework: NestJS v11.2.0
- Database: MySQL 8.0


## Environment Variables

``` dotenv
DB_HOST=localhost
DB_PORT=3306
DB_USERNAME=root
DB_PASSWORD=1234
DB_DATABASE=marsh
```
